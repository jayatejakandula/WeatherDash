# 🌦️WeatherDash – Weather Dashboard

WeatherDash is an interactive, map-based weather dashboard that provides real-time weather insights for any location. It combines geolocation, search functionality, and a dynamic UI to deliver a seamless weather monitoring experience.

---

## 🚀 Features

### 📍 Location-Based Weather
- Click anywhere on the map to get instant weather data
- GPS-based location detection
- Search for any city worldwide

### 🌡️ Real-Time Weather Data
- Temperature (°C)
- Weather condition (Clear, Rain, Clouds, etc.)
- Humidity (%)
- Atmospheric pressure (hPa)

### 🗺️ Interactive Map
- Built using Leaflet.js
- Smooth zooming and panning
- Dynamic marker updates on location selection

### 📊 Forecast (UI Ready)
- Next 6 hours forecast (hourly)
- Next 5 days forecast (daily)
- Scrollable, responsive design

### 🎨 Clean UI/UX
- Minimal and modern interface
- Responsive design for different screen sizes
- Smooth user interactions

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Map:** Leaflet.js  
- **API:** OpenWeatherMap  
- **Deployment:** Vercel  

---

## ⚙️ How It Works

1. User selects a location by:
   - Clicking on the map
   - Searching a city
   - Using GPS

2. The app fetches latitude & longitude

3. Weather data is retrieved from OpenWeatherMap API

4. UI updates dynamically with:
   - Temperature
   - Weather condition
   - Additional metrics
